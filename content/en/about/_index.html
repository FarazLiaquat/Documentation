RoverCore is designed for beginner to intermediate ASP.NET core developers to learn how to develop ASP.NET SaaS applications. The project is a .NET 6 monolithic architecture that incorporates Entity Framework and excludes patterns like CQRS and Repositories that you might find in other starter projects.
